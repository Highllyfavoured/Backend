# Expense Tracker API
The Expense Tracker API is a backend service built with FastAPI, designed to help users manage their expenses securely and efficiently.
It supports user authentication, storing expenses, updating them, deleting them, and retrieving all expenses belonging to a logged-in user.

## This API uses:

FastAPI for backend logic

MySQL (via SQLAlchemy) as the database

JWT authentication for secure access

Bcrypt for password hashing
